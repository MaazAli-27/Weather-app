# 🌤️ Glassmorphism Weather App

A sleek, responsive, and modern weather application built with HTML, Tailwind CSS, and vanilla JavaScript. It features a beautiful glassmorphism design and provides real-time weather data along with a 5-day forecast using the completely free Open-Meteo API.

---

## ✨ Features

* **Real-Time Weather:** Displays current temperature, "feels like" temperature, weather condition description, humidity, and wind speed.
* **5-Day Forecast:** Provides the maximum and minimum temperatures along with weather conditions for the upcoming 5 days.
* **Dynamic Search:** Users can search for almost any city worldwide.
* **Glassmorphism UI:** A modern, frosted-glass aesthetic layered over a dynamic, deep-gradient background.
* **Responsive Design:** Fully optimized for both desktop and mobile devices using Tailwind CSS.
* **Graceful Error Handling:** Includes dedicated loading spinners and error states (e.g., "City Not Found") for a smooth user experience.
* **No API Keys Required:** Utilizes Open-Meteo, which provides free, open-source weather data without requiring user authentication or API keys.

## 🛠️ Technologies & Libraries

* **Frontend Structure:** HTML5
* **Styling:** [Tailwind CSS](https://tailwindcss.com/) (imported via CDN) for rapid, utility-first styling.
* **Logic & Functionality:** Vanilla JavaScript (ES6+ with Async/Await)
* **Icons:** [Lucide Icons](https://lucide.dev/) for crisp, scalable SVG icons.
* **Typography:** [Google Fonts](https://fonts.google.com/) (Inter).
* **Data Sources:** * [Open-Meteo Geocoding API](https://open-meteo.com/en/docs/geocoding-api) (Translates city names to geographic coordinates).
  * [Open-Meteo Forecast API](https://open-meteo.com/en/docs) (Retrieves weather data using those coordinates).
